# 就算是萝莉控也能看懂的Linux常用命令
## 文件和目录操作
### ls
用于列出当前目录的文件和文件夹，常用选项有以下

|选项|作用|
|:---:|:---:|
|`l`|以长格式显示文件和目录的详细信息，包括权限、所有者、大小、修改时间等|
|`a`|显示所有文件，包括隐藏文件（以`.`开头的文件）|
|`h`|以人类可读的格式显示文件大小|

使用`ls -l`后显示如下
```shell
╭─root@kamiw /
╰─# ls -l
total 2120
lrwxrwxrwx   1 root root       7 Apr  8 02:02 bin -> usr/bin
drwxr-xr-x   2 root root    4096 Dec  7  2021 boot
......
```

|列数|意义|示例|解释|
|:---|:---:|:---:|:---:|
|1|文件类型及权限|`lrwxrwxrwx`|第1位表示文件类型，后面9位表示权限|
|2|链接数量|`1`|链接文件为1，普通文件链接数至少为2以上（包含自身及上级目录）|
|3|文件拥有者|`root`|说明该文件所有者为root|
|4|文件所在组|`root`|说明该文件所在组为root组|
|5|文件大小（单位为字节）|`7`|该文件大小为7字节|
|6|修改日期、最后访问日期|`Apr  8 02:02`|表示最后修改日期为4月8日，02时02分|
|7|文件名|`bin -> usr/bin`|对于链接会标记处链接的地址|

### cd
用于切换用户当前目录

使用方法：`cd 路径`

特殊路径

|路径|意义|
|:---:|:---:|
|`..`|返回上级目录（`...`返回上上级目录，以此类推）|
|`/`|根目录|
|`~`|家目录|

### mv
用于移动文件，也可用于更改文件名

使用方法：`mv 旧位置 新位置`

示例（仅移动）：`mv /root/test /home/baka/`  
示例（改名）：`mv /root/test /home/baka/new_test`  
示例（同时移动多个文件）：`mv /root/test1 /root/test2 /root/test3 /home/baka/` 

### cp
用于复制文件，基本方法同`mv`，但是复制目录时需要`-r`

使用方法：`cp 源文件 目标位置`
### rm
用于删除文件，常用选项有：`-r`删除目录；`-f`强制删除不提示或警告；

使用方法：`rm 文件名`
### touch
用于创建文件或更新文件修改时间

使用方法：`touch 文件名`
### mkdir
用于创建目录，常用选项有：`-p`递归创建目录，即如果上级目录不存在，则一并创建;

使用方法：`mkdir 目录名`
### ln
用于创建链接文件，常用选项有：`-s`用于创建软连接

使用方法：`ln 源文件 链接文件`

硬链接：使多个文件名指向同一个数据块，修改一个其他跟着变化，删除其中一个对其他没有影响  
软连接：使链接文件指向源文件，类似于Windows中的快捷方式

## 查看文件内容
### cat
用于查看文件内容

使用方法：`cat 文件名`

|选项|作用|
|:---:|:---:|
|`n`|在每一行前加上行号|
|`v`|将非打印字符以可视化的方式显示|
|`e`|在每行结尾显示`$`符号，用于显示文件的结尾|

### head
用于查看文件内容，默认显示前10行，常用选项`-n`指定显示几行

使用方法：`head -n 行数 文件名`
### tail
用于查看文件内容，默认显示最后10行，常用选项：`-n`指定显示几行；`-f`实时追踪文件的新增内容（常用于查看日志）

使用方法：`tail -n 行数 文件名`

## 简单文本查找与处理
### find
用于在指定目录下查找符合条件的文件和目录(支持通配符与正则)
- 按名称查找：`find 起始目录 -name 文件名`
- 按文件类型查找：`find 起始目录 -type 文件类型`;文件类型：`f`普通文件；`d`目录；`l`链接；
- 按文件大小查找：`find 起始目录 -size 大小`；大小用： `c`字节、`k`千字节、`M`兆字节;`-`、`+`代表大于或小于
- 按时间戳查找，`-atime`访问时间, `-mtime`修改时间, `-ctime`状态改变时间;
  - find <起始目录> -atime +7：查找在过去 7 天内访问过的文件。
  - find <起始目录> -mtime -7：查找在最近 7 天内修改过的文件。
  - find <起始目录> -ctime +7：查找在过去 7 天内状态改变过的文件。
- 可以通过 -and, -or, -not 等逻辑操作符结合多个条件进行复杂的查找
  - 示例（查找`/root`下的`sh`或`conf`文件）：`find /root -name '*.sh' -and -name '*.conf'`


### grep
用于在文件中搜索指定的文本获取关键字所在行,通常也用于过滤管道后的信息

基本使用（文件）：`grep 关键字 文件名`

基本使用（管道）：`command | grep 关键字`；

常用选项

|选项|作用|用法|
|:---:|:---:|:---:|
|`v`|显示不匹配的行|
|`i`|不区分大小写|
|`o`|不显示整行，仅显示匹配部分|
|`n`|显示匹配内容和行数|
|`r`|递归搜索，显示目录下文件中匹配的内容|`grep -r 关键字 目录`|

### cut
用于从文本文件或标准输入中提取文本的列;常用选项有：`f`指定列；`d`指定分隔符；

使用方法：`cut -d 分隔符 -f 列数 文件名`
## 网络
### ifconfig
用于查看网络及修改网络

- 查看网卡及其IP配置：`ifconfig`;使用`ifconfig -a`可查看所有网卡设置；
- 开启网卡：`ifconfig 网卡名 up`
- 关闭网卡：`ifconfig 网卡名 down`
- 修改指定网卡IP：`ifconfig 网卡名 ip地址 netmask 子网掩码`

### ip
功能更为全面的网络配置工具

- 查看网络接口信息：`ip a`
- 查看路由表：`ip r`
- 查看链路状态：`ip l`
- 启用或禁用网络接口
  ```shell
  ip link set dev 网卡名 up
  ip link set dev 网卡名 down
  ```
- 配置及删除网卡IP
  ```shell
  ip addr add IP地址/子网掩码 dev 网卡名
  ip addr del IP地址/子网掩码 dev 网卡名
  ```
- 配置网关
  ```shell
  ip route add default via 网关IP dev 网卡名
  ```
